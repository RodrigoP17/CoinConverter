## Coin Converter 

This is a python project that allows you to convert different coins with real time taxes, through [ExchangeRate API](https://v6.exchangerate-api.com/).

## How to run the project

1. Open a terminal.
2. Go to the project folder:

```bash
cd coin_converter
```

3. Install the dependencies:

```bash
pip install -r requirements.txt
```

4. Execute the app:

```bash
streamlit run app2.py
```

The app will automatically open on your browser.
